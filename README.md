## 🚀 Featured AI Projects

### 🧬 [Skin Lesion Detection Using DenseNet](https://github.com/Ankit1923-mr/Skin_Lesion_Detection_Using_DenseNet)
> Multimodal deep learning pipeline for classifying skin lesions into 7 diagnostic categories

Built a **hybrid CNN + MLP fusion model** combining DenseNet-121 image features with patient metadata (age, sex, localization). Trained on the HAM10000 dataset (10,015 dermoscopic images). Includes **Grad-CAM explainability** visualizations, Albumentations augmentation pipeline, and a deployed frontend.

`PyTorch` `DenseNet-121` `Grad-CAM` `Albumentations` `Jupyter`

---

### 🎭 [Deepfake Audio-Visual Detection API](https://github.com/Ankit1923-mr/deepfake-api)
> Temporal deepfake localization — pinpoints *which seconds* of a video are fake

Novel training-free pipeline using **MediaPipe lip tracking + Whisper phoneme extraction + Dynamic Time Warping** to produce per-second forgery scores (e.g. `[2s–5s: 87% fake]`). Achieves 71.67% accuracy on FakeAVCeleb dataset. Deployed as a FastAPI service on Render.

`FastAPI` `MediaPipe` `OpenAI Whisper` `DTW` `Random Forest` `Docker`

---

### 💰 [FinLingo — Financial LLM via QLoRA](https://github.com/Ankit1923-mr/FinLingo)
> Fine-tuned Llama-3 8B for financial Q&A on a single NVIDIA T4 GPU

Full production-style QLoRA training pipeline using 4-bit quantization + LoRA adapters on the Finance Alpaca dataset. Solved a hardware-specific `bfloat16` bug on T4 GPUs. Ran LoRA rank experiments (r=4,8,16) and tracked results with **Weights & Biases**.

`QLoRA` `PEFT` `Llama-3` `HuggingFace` `BitsAndBytes` `W&B`
